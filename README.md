# Parth Portfolio

My personal portfolio website built using React and Tailwind CSS.

## Features

* Responsive design
* About Me section
* Skills section
* Projects showcase
* Contact section
* Resume download

## Technologies Used

* React
* Tailwind CSS
* JavaScript
* Vite

## Installation

1. Clone the repository

```bash
git clone https://github.com/parthnai21/parth-portfolio.git
```

2. Navigate to the project folder

```bash
cd parth-portfolio
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
